Unleash the Power of AI: Your Ultimate Step-by-Step Guide to Effortless Content Creation

By Shubham Saurav | January 27, 2025

Step-by-Step Guide

  1. Choose a Niche of your liking, you can always do a search on Youtube with this.
  2. Choose a generative AI of your choice it can be ChatGPT, Claude or DeepSeek (Personally not a fan of DeepSeek's narration styles).
  3. If you do not have a topic in mind, just ask AI to generate a random topic list based on the niche you have chosen.
  4. Once you have the topic, just ask AI to generate a script you can use this prompt Generate a script for youtube shorts/instagram reels based on the topic <your_topic_here>.
  5. Once you have the script, you can ask AI to generate images for each scene using this prompt For the script generated above, write prompts for image generation for each scene, make sure the characters are constant in each scene and make sure image generation is <realistic/cartoonish/disney cartoonish>.
  6. Once you have the prompts for Image generation, you can go to Krea.ai website for image generation. You can generate free images as long as you have credits, use tempmailio for temporary mail ids to sign up for infinite credits.
  7. Once you have image generated for each scene, go back to the chat in AI and for each image upload the image to chat and ask to write detailed prompt for video generation With the script above in consideration, this is the image generated for scene<no>. Write a detailed prompt for video generation for this scene..
  8. Once the prompt is there head over to Hailuo.ai .Here if you are using free credits generation will be slow (would suggest to buy one month premium). Now go to text to video option and upload image and paste the prompt. It will generate videos.
  9. Once you have video for each scene head over to Vidmix Website and upload them there. This tool will not add any watermark to your video. It is completely free. Once you have arranged your videos in order added a sound of your choice, export the video. Make sure you select canvas as 9:16 before export.
  10. Voila you have your video ready with you.
